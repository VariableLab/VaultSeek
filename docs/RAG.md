## 二、产品定位（最终定调，不再修改）

本地私有化 AI 语义知识库（本地向量检索系统）

### 核心 Slogan

数据不出本机、断网可用、零 API 费用、小白一键搭建私人智能文档库

## 四、产品整体架构（Demo 最简架构｜可直接开发）

### 整体架构：本地优先 + One-API 云端推理混合架构

**核心设计原则**:
1. **本地优先** - 文档解析、向量化、检索全部在本地完成，数据不出本地
2. **云端补充** - LLM 推理通过 One-API 中转站，每日 10 次免费额度
3. **优雅降级** - 额度用完后本地搜索仍可用，不阻断用户体验
4. **双轨付费** - 官方额度充值 OR 自带 API Key 无限使用

### 架构组件:

1. **前端界面**：Electron + React (Notion 风格黑白灰极简设计)
   - 左侧导航 (240px): 知识库/文档管理/智能问答/历史记录
   - 顶部导航 (48px): 面包屑/通知/用户菜单
   - 主内容区：统计卡片/文档列表/对话界面

2. **本地向量库**: Chroma (轻量版，无需额外服务)
   - 文档元数据：SQLite 存储
   - 向量索引：Chroma 本地管理
   - 支持断点续传、进度可视化

3. **本地 Embedding**: @xenova/transformers (轻量离线模型)
   - 768 维向量
   - 适配低配电脑，无需 GPU

4. **云端推理**: One-API 中转站
   - 每日 10 次免费额度
   - 超出后弹窗引导付费 (198 终身/68 年费)
   - 支持自带 API Key 无限使用

5. **文件解析模块**: PDF/Word/TXT 本地解析
   - pdf-parse (PDF)
   - mammoth (Word)
   - 原生读取 (TXT)

### 核心优势（写入产品介绍）

- 100% 数据本地存储，**零数据外传**

- 无需联网使用

- 无需独立显卡，办公本流畅运行

- 无需任何编程环境、无需 Docker

## 五、Demo 版本功能清单（最小可行产品，AI 直接开发）

### 模块 1：文档导入模块

1. 支持批量导入：PDF / Word / TXT

2. 自动切片、自动清洗基础格式

3. 导入进度可视化

4. 本地建立专属知识库文件夹

### 模块 2：本地向量化模块

1. 本地离线生成向量，不调用任何云端接口

2. 自动去重、自动精简冗余内容

3. 向量库本地加密存储

### 模块 3：智能语义问答核心（产品灵魂）

1. 自然语言提问

2. AI 基于本地所有文档回答

3. 回答附带：**来源文档 \+ 页码 / 段落定位**（核心碾压竞品功能）

4. 支持总结、归纳、对比、提炼知识点

### 模块 4：权限与额度系统（商业变现核心）

1. 免费版机制

- 每日免费 10 次问答

- 最大可导入文档 50 份

2. 付费版解锁全部限制

### 模块 5：基础 UI 系统（Notion 风格设计）

**设计风格定义**
- 配色：黑白灰极简配色（主背景 `#ffffff`、侧边栏 `#f7f7f5`、文字 `#37352f`、边框 `#e9e9e7`）
- 圆角：`8px`（卡片）、`4px`（小元素）
- 阴影：极淡或无阴影，内容优先
- 按钮：纯黑背景 + 白色文字，悬停轻微透明度变化
- 图标：功能驱动，简洁表意

**界面布局**
1. 左侧固定导航栏（240px 宽）：知识库入口、文档管理、智能问答、历史记录
2. 顶部导航栏（48px 高）：面包屑、通知、用户菜单
3. 主内容区：统计卡片、文档列表/网格、问答对话框

**核心界面**
- 控制台：4 个统计卡片 + 最近文档表格
- 文档管理：网格卡片视图 + 类型筛选
- 智能问答：对话流 + 来源引用标签
- 空状态页：引导导入 + 支持格式说明

## 八、产品禁止逻辑（开发必须严格遵守｜避坑核心）

1. **禁止宣传 100% 全文档通杀**
产品文案固定：适合电子办公文档，复杂图文存在正常误差

2. **禁止做重度企业协同 Demo**

3. **禁止接入任何云端自动上传逻辑**

4. **禁止使用按量计费 API**

5. **Demo 阶段不开发复杂 OCR、不开发复杂权限**

## 九、产品核心卖点固定（全部适配市场真实需求）

1. 真本地、真离线、真不外泄

2. 无 API 隐形消费、终身使用划算

3. 低配电脑流畅运行

4. 语义精准，远超系统关键词搜索

5. 小白一键安装，零技术门槛

6. 资料溯源定位，办公效率质变提升
