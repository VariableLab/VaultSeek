# 更新日志 (CHANGELOG.md)

## [1.2.0] - 2026-06-06
### 🚀 新增
- 引入 Spotlight 交互模式：`Alt + S` 全局唤起。
- 引入三栏深色模式对话工作台。
- 引入基于本地知识库的混合检索 + LLM 问答功能。
- 引入文档解析高精度流水线（支持 Excel/Docx/PDF 结构化 MD 转换）。
- 集成 `keyring` 实现 API Key 安全加密存储。

### 🛠 优化
- 性能优化：后端采用多线程并行处理 (Rayon)。
- UI 交互：增加自定义窗口控制栏，优化窗口缩放响应式布局。
- 检索算法：实现关键词命中权重算法，过滤低置信度噪音。
