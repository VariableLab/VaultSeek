# VaultSeek 核心架构与商业蓝图 (MASTER BLUEPRINT)

**状态**: Active / v2.0 规划中
**机密级别**: 内部核心 (Confidential)

---

## 1. 商业定位与护城河 (The Business Playbook)

### 1.1 核心价值观
VaultSeek 是一款**大众通用型知识加速器**。通过“本地私有计算”与“动态知识画像”，将杂乱的个人文档变为懂你的私人 AI 智囊。

### 1.2 核心护城河 (Adaptive Intelligence)
*   **主动式智能画像系统 (Proactive Persona System)**：VaultSeek 不仅是检索工具，它是一个通过观察用户数据流（文档结构、搜索历史）自动构建“用户知识偏好向量”的 AI。它能理解用户是法务、医生还是开发者，并在问答中自动补全该行业的隐含逻辑。
*   **意图超前评估 (Intent Look-ahead)**：基于“意图重写流水线”，系统在检索 Token 前即判断用户需求的合理性与完整性。通过主动反问、诊断，从“被动检索”升维为“主动顾问”，彻底区别于传统的 Chatbot。

---

## 2. 系统技术架构 (The Architecture)

### 2.1 智能流水线 (The Pipeline)
1. **意图诊断 (Intent Layer)**：用户输入 -> 模型诊断意图 -> (如意图模糊) -> 主动反问/扩充关键词。
2. **混合检索 (Retrieval Layer)**：基于改写后的精准关键词与语义向量，捞出核心 1000 Token 资产。
3. **合成层 (Synthesis Layer)**：组装“系统 Prompt + 画像向量 + 资产 Token”，生成比用户所想更全面、更严谨的报告。
