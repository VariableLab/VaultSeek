# 未来扩展策略 (EXPANSION_STRATEGY.md)

## 1. 短期 (v2.x)
- **多知识库支持**: 实现工作空间切换逻辑。
- **高阶工作流**: 引入跨库数据对比与可视化分析。

## 2. 中期 (v3.0 - 生态化)
- **Prompt Marketplace**: 实现 Prompt 模板的本地增量下载与用户共享。
- **Agentic Actions**: 允许用户定义“动作”，例如检索完后一键发送邮件或创建 Obsidian 笔记。

## 3. 远期 (v4.0 - 智能化)
- **端侧模型微调 (Fine-tuning)**: 允许用户在本地对小模型进行微调，让它更懂特定行业的术语。
- **多模态解析**: 引入 OCR 视觉解析引擎，支持扫描件 PDF 的深度理解。
